Python GUI => PyQt (Desktop Applications)

1. pip install PyQt5
2. download and install pyqt gpl (it is the pyqt designer)
3. create and save the design from designer with .ui extension
4. write command to convert .ui file into .py file
   "python -m PyQt5.uic.pyuic -x filename.ui -o filename.py"
